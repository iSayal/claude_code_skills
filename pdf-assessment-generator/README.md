# PDF Assessment Generator Skill

This skill analyzes PDF documents and generates comprehensive 20-question MCQ assessments to test user knowledge. It works with academic textbooks, training materials, and technical documentation, creating questions ranging from basic to advanced difficulty levels based solely on the PDF content.

## Overview

The PDF Assessment Generator skill enables users to:
- Upload PDF documents containing educational content
- Automatically generate 20 multiple-choice questions based on the content
- Receive questions distributed across difficulty levels (basic, intermediate, advanced)
- Obtain an answer key for the generated assessment

## Components

- `SKILL.md`: Main skill definition and usage instructions
- `references/`: Supporting documentation on PDF processing and assessment techniques
- `scripts/`: Utility scripts for PDF analysis and question generation

## Usage

1. Provide a PDF file containing educational content (textbooks, training materials, technical documentation)
2. The skill will analyze the content and generate 20 MCQs
3. Questions will be distributed across difficulty levels based on content complexity
4. An answer key will be provided for assessment scoring

## Requirements

- PDF processing capabilities
- Natural language processing for content analysis
- Question generation algorithms
